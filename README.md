# Todo App

A simple **JavaScript Todo App** to add and delete tasks. The app saves tasks in the browser, so they stay even after refreshing.

---

## Features

- Add tasks
- Delete tasks
- Save tasks in browser (localStorage)
- Simple and clean design

---

## How to Use

1. Open `index.html` in browser.
2. Type a task in the input box.
3. Click **Add Todo** to save the task.
4. Click **Delete** to remove a task.
